# Numericals

Numerical methods in pure Python.

Still very much in development.

## Installation
```pip install numericals```

## Algorithms
- Integration
    - Trapezoidal Rule
    - Midpoint Rule
    - Simpson's Rule
    - Monte Carlo

- ODE
    - Euler's Method

- Optimization
    - Golden Section Search

- Root-finding
    - Bisection Method
    - Secant Method
